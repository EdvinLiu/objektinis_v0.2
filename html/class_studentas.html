Detailed Description

Klasė, apibūdinanti studentą, paveldinčią Zmogus klasę.

Studentas klasė turi papildomus duomenis, tokius kaip egzaminų ir namų darbų pažymiai, bei metodus, skirtus manipuliuoti šiais duomenimis.
